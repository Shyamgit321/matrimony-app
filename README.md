# Matrimony App (MySoul)

A full-stack matrimony web app.

## Tech Stack

* Frontend: HTML, CSS, JS
* Backend: Node.js, Express
* Database: MongoDB

## Features

* Login (Email + Google)
* Profile creation & edit
* Chat system
* Interest system
* Privacy settings

## Run locally

cd Back-End
npm install
npm start
